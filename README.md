# Scrapping-puppeter
web scrapping application with puppeter and lazy loading implementation

## Local Environment Setup

Ensure you're using Node v8 or higher

    npm install

Run Server

    npm start

## Node Package

Puppeter package

    https://www.npmjs.com/package/puppeteer
